# Functional Requirements (FRs)

## FR1: Browse Available Meals
As a user, I want to browse all meals offered by the catering service so that I can see what options are available.

## FR2: View Meal Ingredients
As a user, I want to view the complete list of ingredients needed for each meal so that I know what I need to buy.

## FR3: Search Meals
As a user, I want to search for meals by name, ingredient, or category like vegan or gluten-free so that I can quickly find meals that match my needs.

## FR4: Order Ingredients
As a user, I want to order ingredients for a selected meal so that I can get everything delivered to prepare it.

## FR5: Schedule Delivery
As a user, I want to choose between same-day or next-day delivery so that I can get ingredients when it works for my schedule.

## FR6: Track Orders
As a user, I want to see the current status of my orders so that I know when my ingredients will arrive.
